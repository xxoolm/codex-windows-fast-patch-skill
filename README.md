# Codex Windows Fast Patch Skill

[![LINUX DO](https://img.shields.io/badge/LINUX_DO-community-00aeef?style=flat-square)](https://linux.do/)

Community link: [LINUX DO](https://linux.do/)

Public, sanitized export of the `codex-windows-fast-patch` skill.

This skill documents and automates a Windows Codex Desktop repatch workflow, including:

- Reapplying the Windows MSIX patch after Codex Desktop upgrades.
- Verifying Fast Mode request wiring.
- Registering local plugin marketplaces.
- Repairing local marketplace manifest layout.
- Refreshing Windows Computer Use compatibility files.
- Unlocking the Windows Computer Control "Any App" gate when the UI reports organization or region unavailability, by patching the local Desktop webview gates and enabling `features.computer_use`.

## Included Files

- `SKILL.md`
- `agents/openai.yaml`
- `scripts/repatch-codex-windows.ps1`
- `scripts/patch_codex_fast_mode_windows_msix.ps1`
- `scripts/install-computer-use-local.ps1`

## Install

Copy this repository into your Codex skills directory:

```powershell
Copy-Item -Recurse . "$env:USERPROFILE\.codex\skills\codex-windows-fast-patch"
```

Restart Codex after installing or updating the skill so the runtime reloads skill metadata.

## Sanitization

This repository intentionally excludes local Codex state, authentication files, logs, plugin caches, generated bytecode, dependency folders, and machine-specific configuration.

Before publishing, this export was scanned for common secret and local-path patterns such as API keys, GitHub tokens, local user-profile paths, `auth.json`, `config.toml`, and private endpoint strings.
